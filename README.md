# ⚡ Projeto Shape Aesthetic (V-Taper & Pernas Grossas)

Este repositório funciona como o meu **QG de Evolução Física**, centralizando o planejamento de treinos de alta intensidade e a estratégia nutricional para ganho de massa magra. O projeto é otimizado para o ecossistema do **Claude AI** e distribuído via **GitHub Pages** para acesso rápido pelo celular.

## 🎯 Objetivos do Projeto
*   **Físico:** Construção de um shape *V-Taper* (dorsais largas e ombros destacados) combinado com membros inferiores densos (quadríceps, glúteos e panturrilhas).
*   **Nutrição:** Adaptação de macronutrientes "no olhômetro" baseada na realidade diária do Restaurante Universitário (RU).
*   **Automação:** Registro limpo de cargas e deploy automatizado utilizando scripts em Python que seguem rigorosamente os padrões **PEP8** e **PEP257**.

## 🛠️ Tecnologias e Ferramentas
*   **Frontend:** HTML5, Tailwind CSS (Design System Premium) e Marked.js.
*   **Backend/Automação:** Python 3 (Scripts de deploy e logs).
*   **AI Engine:** Claude AI (como Personal Trainer de Elite através de Contexto de Projeto).
